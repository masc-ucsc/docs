    Pyrope Standard Library¶

    This is a list of functionality that import prp should produce.

    Basic operations¶

    All the LNAST node have an associated function matching name to simplify the creation of operations: plus, minus, mult, div, mod, ror...

    const prp = import("prp")
cassert prp.plus(1,2,3) == 6

    Library code: 

    const plus = comb(...a:int)->(_:int) {
  mut r = 0
  for e in a {
    r += e
  }
  r
}

    Array/Tuple operators¶

    Size of length¶

    Sample use: 

    const x = (1,2,23)

cassert p.len(x) == 3

    Library code: 

    const len = comb(x) { x::[size] }

    map¶

    Sample use:

    const x = (1,2,3)

cassert x.map(fun(i){ i+1 }) == (2,3,4)

    Library code: 

    const map = fun<T>(f:fun(a:T),...x:[]T) {
  return f(e) for e in x
}

    filter¶

    Sample use:

    cassert (1,2,3).filter(fun(i){ i!=2 }) == (1,3)

    Library code:

    const filter = fun<T>(f:fun(a:T)->(_:Bool),...x:[]T) {
  return e for e in x if not f(e)
}

    reduce¶

    Sample use:

    cassrt (1,2,3).reduce(prp.plus) == 6

    Library code:

    const reduce = comb(op:fun<T>(a:T,b:T)->(_:T), ...x) {
  return x when x::[size] <= 1

  mut res = x[0]
  for i in x[1..] {
    res = op(res, i)
  }
  return res
}
